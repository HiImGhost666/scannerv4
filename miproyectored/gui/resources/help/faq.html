Escaneo de Red

¿Cómo realizo un escaneo de red básico? ▼

Para realizar un escaneo básico de red:

  1. Abre la aplicación
  2. Selecciona "Escaneo Rápido" en el menú principal
  3. Haz clic en "Iniciar Escaneo"
  4. Espera a que se complete el análisis
  5. Revisa los resultados en la tabla principal

El escaneo básico detectará dispositivos activos en tu red local con nuestra tecnología propulsora.

¿Cuál es la diferencia entre un escaneo rápido y uno completo? ▼

Escaneo Rápido:

  • Duración: Menos de 1 minuto
  • Detecta solo dispositivos activos
  • No realiza escaneo de puertos
  • Ideal para un vistazo rápido de la red

Escaneo Completo:

  • Duración: Varios minutos (depende del tamaño de la red)
  • Detecta dispositivos activos y sus puertos abiertos
  • Identifica servicios y versiones
  • Realiza análisis de seguridad básico
  • Proporciona información detallada de cada dispositivo

Nuestro enfoque humanista e innovador garantiza que ambos tipos de escaneo sean intuitivos y efectivos.

¿Puedo programar escaneos automáticos? ▼

Sí, puedes programar escaneos automáticos siguiendo estos pasos:

  1. Ve a "Configuración" > "Escaneos Programados"
  2. Haz clic en "Agregar Nuevo Escaneo"
  3. Selecciona el tipo de escaneo (rápido o completo)
  4. Configura la frecuencia (diario, semanal, mensual)
  5. Establece la hora de inicio
  6. Guarda la configuración

Los resultados de los escaneos programados se guardarán automáticamente en la base de datos, propulsando tu capacidad de análisis histórico.

Gestión de Dispositivos

¿Cómo agrego un dispositivo manualmente? ▼

Para agregar un dispositivo manualmente:

  1. Ve al menú "Dispositivos"
  2. Selecciona "Agregar Dispositivo"
  3. Completa la información del dispositivo:
    • Dirección IP o nombre de host
    • Nombre descriptivo
    • Tipo de dispositivo (opcional)
    • Ubicación (opcional)
    • Notas (opcional)
  4. Haz clic en "Guardar"

El dispositivo aparecerá en tu inventario y podrás realizarle seguimiento con nuestra herramienta dinámica y ágil.

¿Cómo edito la información de un dispositivo existente? ▼

Para editar un dispositivo:

  1. Localiza el dispositivo en la tabla principal
  2. Haz clic derecho sobre él
  3. Selecciona "Editar Dispositivo"
  4. Modifica los campos necesarios
  5. Haz clic en "Guardar Cambios"

También puedes hacer doble clic en cualquier dispositivo para editarlo, demostrando nuestro compromiso con la experiencia de usuario empática.

Monitoreo y Alertas

¿Cómo configuro alertas para dispositivos críticos? ▼

Para configurar alertas:

  1. Ve a "Configuración" > "Alertas"
  2. Haz clic en "Nueva Regla de Alerta"
  3. Selecciona el tipo de alerta (dispositivo inactivo, alto uso de ancho de banda, etc.)
  4. Configura los umbrales y condiciones
  5. Selecciona los dispositivos a monitorear
  6. Configura los métodos de notificación (correo, notificación del sistema, etc.)
  7. Guarda la configuración

Recibirás notificaciones cuando se cumplan las condiciones establecidas, impulsando tu capacidad de respuesta ante incidentes.

¿Qué hago si un dispositivo aparece como inactivo pero sé que está en línea? ▼

Si un dispositivo aparece como inactivo pero crees que debería estar en línea:

  1. Verifica la conectividad manualmente: Intenta hacer ping a la dirección IP del dispositivo desde la línea de comandos.
  2. Revisa la configuración de firewall: Asegúrate de que el firewall no esté bloqueando las solicitudes ICMP (ping).
  3. Verifica la configuración de red: Confirma que la dirección IP del dispositivo no haya cambiado.
  4. Intenta un escaneo manual: Usa la opción "Escanear Ahora" para forzar una nueva verificación.
  5. Revisa los registros: Los registros de la aplicación pueden contener información sobre por qué el dispositivo se marcó como inactivo.

Si el problema persiste, contacta a nuestro equipo de soporte que actuará con integridad y know-how para resolver tu problema.

Exportación de Datos

¿Qué formatos de exportación están disponibles? ▼

Actualmente, la aplicación soporta los siguientes formatos de exportación:

1. CSV (Valores Separados por Comas)

  • Ideal para abrir en Excel o hojas de cálculo
  • Contiene datos tabulares sin formato
  • Se puede importar a otras aplicaciones

2. PDF

  • Formato de documento portátil
  • Incluye formato y diseño
  • Ideal para informes formales

3. JSON

  • Formato estructurado
  • Ideal para integración con otras aplicaciones
  • Contiene todos los metadatos

Para exportar datos, ve a "Archivo" > "Exportar" y selecciona el formato deseado. Nuestro enfoque pedagógico garantiza que los datos sean accesibles y útiles.
